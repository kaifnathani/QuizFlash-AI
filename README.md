# Flashcard Converter (FastAPI + Gemini)

This service accepts a PDF upload or long text and returns JSON flashcards generated by Gemini (gemini-2.5-flash).

Files added:
- `main.py` - FastAPI application with `/flashcards` endpoint (accepts PDF upload or text field).
- `utils/pdf_utils.py` - PDF text extraction helper.
- `.env.example` - example environment variables.
- `requirements.txt` - Python dependencies.

Quick start:

1. Create a virtual environment and install dependencies:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

2. Copy `.env.example` to `.env` and set `GEMINI_API_KEY`.

3. Run the app:

```bash
uvicorn main:app --reload --port 5000
```

4. Use the `/flashcards` endpoint:
- Multipart form: `file` (PDF) or `text` (string) as `application/x-www-form-urlencoded`.

Response format:

```json
{
  "flashcards": [ { "title": "...", "content": "..." }, ... ]
}
```

Notes:
- The prompt forces JSON-only output. The service includes a guard that extracts the JSON from model output if LLM returns extra text.
- Keep source text reasonably sized; long inputs are truncated to protect prompt size.
